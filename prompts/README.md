# Frozen 12-prompt pool

Four categories × three prompts. A future public beacon selects one prompt from each category.

| Category | Purpose |
|---|---|
| A | intent and product classification |
| B | state and workflow correctness |
| C | authorization and isolation |
| D | change resilience |

The machine source of truth is `pool.json`. Prompt text is exact and must be pasted verbatim for every entrant.

The headliner, A1, is intentionally the prompt that previously broke WholeStack. Publishing that fact is part of the credibility model.
